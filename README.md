4. Подробное описание основной фишки — Система репутации
gdscriptclass_name ReputationManager extends Node

enum Faction { RUSSIA, INDIGENOUS, ECOLOGISTS, CHINA, WEST }

var reputation: Dictionary = {} # Faction → int (-100..100)
var prestige: int = 0
var reputation_points: int = 100

func change_reputation(faction: Faction, delta: int, reason: String):
    reputation[faction] = clamp(reputation[faction] + delta, -100, 100)
    EventBus.emit_signal("reputation_changed", faction, reputation[faction], reason)

5. Климатическая система и динамический лед

Температура повышается на 0.04–0.07 °C за ход (настраивается).
Расчёт площади и толщины льда — каждый ход на GPU (RenderingDevice Compute Shaders).
Лед влияет на судоходство по СМП, добычу, строительство.

6. Форматы данных и моддинг

Баланс, здания, технологии — Godot Resource (.tres) + JSON для модов.
Сохранения — бинарный MessagePack + сжатие Zstandard.
Моды: папка mods/, поддержка .pck и нативных GDExtension-модулей.

7. Графика и стиль

Low-poly модели + PBR-материалы.
Кастомные шейдеры снега/льда с динамическим отражением.
Пост-обработка: морозный эффект, полярное сияние, метель.
UI с эффектом инея на стекле.
